# simulation (멀티에이전트 시뮬레이션)

본 서비스와 별개로 도는 **독립 정적 HTML** 시뮬레이션입니다. 여러 에이전트의 협업을 보여줍니다.

| 파일 | 역할 |
|---|---|
| `agent_simulation.html` | 시뮬레이션 화면. 브라우저로 직접 엽니다. |
| `sim_modules/mod_asset.js` | 자산 부문 모듈 |
| `sim_modules/mod_docs.js` | 서류행정 부문 모듈 |
| `sim_modules/mod_fraud.js` | 사기탐지 부문 모듈 |
| `sim_modules/CONTRACT.md` | 시뮬레이션 모듈 규약 |

백엔드 없이 단독으로 돕니다. 발표나 데모에서 아키텍처를 시각적으로 보여줄 때 씁니다.
