# 제출결과물 (데이콘 본선 최종 제출물)

팀 **폴인포린**이 데이콘 본선에 제출한 **최종 결과물**입니다. 모두 최신 확정 버전입니다.

## 파일

| 파일 | 종류 | 역할 | 버전 |
|---|---|---|---|
| `MVP_폴인포린.pdf` | PDF | MVP 제안서 최종본 | 2026-06-11 |
| `기능명세서_외국인_라이프케어_AI_Agent_최종.docx` | DOCX | 기능 명세서 최종본 | v4 (박시현 수정) |
| `[데이콘] ... 발표자료_..._버전2_동우수정.pptx` | PPTX | 발표 자료 최종본 (20슬라이드) | v2 (곽동우 수정) |
| `기능명세서_폴인포린.pdf` | PDF | 기능명세서 시각 요약본 | - |
| `발표자료_외국인_라이프케어_AI_Agent.html` | HTML | 발표 자료 웹 버전 백업 | - |
| `폴인포린-Cover.jpg` | JPEG | 팀 표지 이미지 | - |
| `JB그룹_폴인포린_시연영상.mp4` | MP4 | 시연 영상 | 깃 제외 |

## 중요

- **시연영상(.mp4)은 깃에 포함되지 않습니다.** `.gitignore`로 제외했습니다.
  데이콘에는 직접 업로드하고, 제출 압축본을 만들 때는 이 파일을 수동으로 넣습니다.
- 제출 압축본은 상위 `liferoad_ai_agent/` 폴더 전체를 압축합니다(코드 + 이 결과물).
- 구버전(`.bak`)과 미압축 원본(`.mov`)은 정리하며 삭제했습니다. 여기 있는 것이 전부 최종본입니다.
