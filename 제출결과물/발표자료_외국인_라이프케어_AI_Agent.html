03-B제안 솔루션 개요 - 판단, 행동, 검증 루프
①  판단
LLM이 질문하고, 페르소나를 해석해 호출할 도구와 순서를 결정
Rule Engine - 전세가율 80% 등 법령 상수 + 6대 사기 룰 판단
Isolation Forest - 국적 × 비자 그룹별 이상치를 점수화
D-Day 엔진 - 마감 임박도로 우선순위 산출
→
②  행동
도구 연쇄 호출로 근거 수치 조회, 재해석
마감, 절감, 위험 알림 카드 푸시
송금 경로 수정안 제시
정부 신청서 PDF 자동작성, 모국어 본인확인 발송
→
③  검증 및 개선
루프 종료 조건(stop_reason), 실패 시 재시도
OCR 신뢰도 0.85 미만 → 수동검토(Human-in-the-loop)
구조화 JSON 2차 정합성 검증(Self-Correction)
계약 가드레일 테스트 자동 검사, 큰 결정은 본인 승인
↻  판단 → 행동 → 검증, 개선이 단발이 아니라 하나의 멀티턴 루프로 순환한다
